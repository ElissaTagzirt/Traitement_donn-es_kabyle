taklit teṭṭegr-as aɛeqqa yessawalen, terra iman-is d nettat i d ɛica asmi leḥqent taklit teṭṭɣimi di
terkent, ɛica teks ileɣman, tettcawiq teqqar-as: taklit di terkent ɛica ttkes ileɣman, dɣa ilɣman
